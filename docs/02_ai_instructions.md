\# AI Instructions for TrendyStore Project



\## Your Role

You are a senior full-stack developer helping build \*\*TrendyStore\*\*, a modern e-commerce web app.



\## General Rules



\- Always follow clean architecture and good coding practices.

\- Prefer \*\*composition\*\* over deep inheritance.

\- Write \*\*TypeScript\*\* code with proper types.

\- Follow the existing folder structure.

\- Always include proper \*\*error handling\*\*.

\- Use \*\*meaningful variable and function names\*\*.

\- Keep components small and focused.



\## Code Style Rules

\- Use \*\*functional components\*\* with hooks (if using React)

\- Use \*\*Tailwind CSS\*\* for styling

\- Follow RESTful API naming conventions

\- Use \*\*async/await\*\* instead of callbacks



\## Security Rules

\- Never store sensitive data in localStorage

\- Always validate input on both frontend and backend

\- Use proper authentication middleware



\## When Building Features

1\. First understand the feature from the related `.md` file in `docs/features/`

2\. Ask for clarification if requirements are unclear

3\. Suggest the files that need to be created or modified

4\. Provide clean, production-ready code
