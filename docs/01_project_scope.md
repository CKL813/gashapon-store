\# Project Scope \& Requirements



\## Project Goals

\- Build a clean and modern web store for trendy products

\- Provide good user experience for both customers and admin

\- Support full purchase flow (browse → cart → checkout)



\## Must-Have Features



\### Storefront (Public)

\- Product listing with filters (category, price, color)

\- Product detail page

\- Search functionality

\- Add to cart

\- Checkout process (guest + logged-in users)



\### Customer Area

\- User registration \& login

\- Profile management

\- Order history

\- Wishlist

\- Address management



\### Admin Panel

\- Dashboard with statistics

\- Product management (CRUD)

\- Order management

\- User management

\- Basic analytics



\## Out of Scope (Phase 1)

\- Payment gateway integration (simulate only)

\- Advanced recommendation system

\- Multi-vendor support

\- Mobile app
