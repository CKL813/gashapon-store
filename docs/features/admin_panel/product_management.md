\# Feature: Product Management (Admin)



\## Description

Admin can create, read, update, and delete products.



\## Requirements

\- Add new product (name, description, price, images, category, stock)

\- Edit existing product

\- Delete product

\- View list of all products

\- Search and filter products in admin panel



\## Admin Flow

1\. Admin logs into admin panel

2\. Goes to "Products" section

3\. Can see all products in a table

4\. Can add/edit/delete products



\## Fields Required

\- Product name

\- Description

\- Price

\- Category

\- Images (multiple)

\- Stock quantity

\- Status (Active / Inactive)
