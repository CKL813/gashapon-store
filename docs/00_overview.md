\# TrendyStore - Project Overview



\*\*TrendyStore\*\* is a modern e-commerce web application for selling trendy fashion products (clothing, accessories, shoes, etc.).



\## Target Users

\- Customers (browsing and buying products)

\- Registered users (with accounts and order history)

\- Admin (managing products, orders, and users)



\## Main Modules

\- \*\*Storefront\*\*: Public product browsing and purchasing

\- \*\*Customer Area\*\*: User account, orders, wishlist

\- \*\*Admin Panel\*\*: Product \& order management



\## Tech Goals

\- Fast and clean user experience

\- Mobile responsive

\- Secure authentication

\- Easy product management for admin
